ABOUT

Hi! My name is Elisa Luo and I am a self-taught high school student interested in learning more about technology and programming. I have made many websites for fun and has also experimented with Android Studio in making Android apps. I am always eager to try out new things and learn from my mistakes!


I started my coding journey by learning to code web pages with HTML & CSS at the age of 12, and my interests towards programming grew only from there on. Over the past three years, I have been experimenting with many different platforms in creating different applications. Recently, I also began to devote my time towards circuits & single-board computers to try to use my programming skills in the real physical world.


Some cool things I can do include:

  • Making websites using HTML, CSS, Javascript, Semantic UI, Bootstrap, MongoDB, Express.js, Angular, Vue.js, React.js, and Node.js
  • Creating simple Android apps using Java & Android Studio
  • Building websites with Oracle Application Express
  • Creating & managing databases with SQL
  • Programming FRC robots using Java
